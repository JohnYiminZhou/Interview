Push local project to remote.

1. Create an empty respository.

2. touch README.md in local file

3. git init

4. git commit -m

5. git remote add origin (...)
Connect remote respository

6. git push -u origin master

Save changes and push to remote respository:

1. git add .
ready to submit

2. git commit -m
save change to local

3. git push
push to remote respository

4. git pull
copy source code from remote respository
